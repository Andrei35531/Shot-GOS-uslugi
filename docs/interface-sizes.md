# Размеры интерфейса (шрифты, отступы, карточки)

## Базовые настройки

| Переменная | Значение | Где используется |
|------------|----------|------------------|
| `--font-size` | 16px | База в `:root` (theme.css) |
| Шрифт интерфейса | **Golos Text** | DocumentCard, MobileHeader |

## Шрифты

| Элемент | Размер | Начертание | Межстрочный интервал | Файл/компонент |
|---------|--------|------------|----------------------|----------------|
| Заголовок страницы (h1 в хедере) | **17px** | medium (500) | 1.5 (leading-6) | MobileHeader.tsx |
| Заголовок карточки документа | **18px** | medium (500) | 1.25 | DocumentCard.tsx |
| Номер документа (подзаголовок) | **16px** (text-base) | normal (400) | по умолчанию | DocumentCard.tsx |

| Базовые h2 | var(--text-xl) | medium | 1.5 | theme.css |

## Отступы

### Список документов (DocumentListPage)

| Константа / класс | Значение | Описание |
|-------------------|----------|----------|
| `PADDING_TOP` | **16px** | Отступ сверху контента |
| `CARD_GAP` | **4px** | Расстояние между карточками |
| `PEEK_PX` | **88px** | Высота видимой полоски предыдущей карточки в стаке |
| `bottomPadding` | вычисляется (min 120px) | Нижний отступ контента |
| `bottom-2` | **8px** | Отступ индикатора снизу |
| `px-4 py-4` (header) | **16px** по горизонтали и вертикали | Отступы шапки |

### Шапка (MobileHeader)

| Класс / стиль | Значение |
|---------------|----------|
| `px-4 py-4` | 16px слева/справа, 16px сверху/снизу |
| Кнопки назад/меню | 40×40px (`w-10 h-10`) |
| Иконки в кнопках | 28×28px (ChevronLeft), 24×24px (MoreVertical) |

### Карточка документа (DocumentCard)

| Класс / стиль | Значение |
|---------------|----------|
| `p-6` | **24px** внутренний отступ со всех сторон |
| `mb-2` (под заголовком) | **8px** отступ между заголовком и номером |
| Иконка: отступ от краёв | right: **15px**, bottom: **15px** (подложка); right: **31px**, bottom: **31px** (круг иконки) |

### Страница документа (DocumentDetailPage)

| Элемент | Значение |
|---------|----------|
| Padding канваса | **8px 12px** (top/bottom 8px, left/right 12px) |
| Высота карточки/канваса | **400px** |
| Скругление области документа | **0.75rem** (12px) |

### Мокап (MockupFrame)

| Параметр | Значение |
|----------|----------|
| Ширина viewport контента | **412px** |
| Скругление экрана в мокапе | **24px** |
| Скругление рамки мокапа (Bezel) | **48px** (rounded-[48px]) |
